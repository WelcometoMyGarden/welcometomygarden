# Translation guidelines

## General

### Tone of voice

- **Warm and personal.** Generally, the platform speaks directly to users in the second person with informal, personal pronouns (see language-specific exceptions below).
- **Community-minded.** WTMG is a citizen initiative; the copy reflects solidarity and a shared spirit.
- **Light and encouraging.** Rules and instructions are framed positively.

### The following words should never be translated

- WTMG
- Welcome To My Garden
- Superfan
- Slowby

## Per-language guidelines

Each language contains a glossary on top. Generally, derived or related forms of these words should also be translated with their equivalent form in other languages, except if explicitly pinned in the dictionary. E.g.: slow traveller (en) becomes trage reiziger (nl), but membership becomes adhésion (fr).

### Dutch (nl)

| English (en) | Dutch (nl)   |
| ------------ | ------------ |
| slow travel  | traag reizen |
| host         | host         |
| member       | lid          |
| membership   | lidmaatschap |

- **Informal, familiar tone**: In Dutch, we use the informal _je/jij_ (never _u_).

### French (fr)

| English (en) | French (fr) |
| ------------ | ----------- |
| slow travel  | voyage lent |
| host         | hôte        |
| member       | membre      |
| membership   | adhésion    |

- **Respectful language**: In French, using `vous` instead of `tu`
- **Inclusive language**: The French translation uses inclusive writing with mid-dot notation: _voyageur·se·s_, _citoyen·nes_, _passionné·es_. Apply the inclusive-writing convention appropriate to the target language if one exists.

### German (de)

| English (en) | German (de)      |
| ------------ | ---------------- |
| slow travel  | langsames Reisen |
| host         | Gastgeber        |
| member       | Mitglied         |
| membership   | Mitgliedschaft   |
| camping      | Zeltplatz        |

- DON'T use "Campingplätze" or "camping" in translations
- Keep the following terms in English if they were in English, but capitalize them German-style: Slow Traveller, User, Host
- **Use informal personal pronouns and tone** (e.g. du, not Sie)
